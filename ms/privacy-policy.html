Dasar Privasi dan Pengurusan Data untuk "Mega Factory Tycoon: Grid Pengeluaran"

Kemas Kini Terakhir: [17.10.2025]

Dasar privasi ini menerangkan bagaimana aplikasi mudah alih "Mega Factory Tycoon: Grid Pengeluaran" mengumpul, menggunakan, dan mengurus data pengguna.

1. Data yang Kami Kumpul

Aplikasi kami hanya mengumpul data terhad yang diperlukan untuk menyampaikan dan meningkatkan pengalaman permainan.

  • Data Permainan Tempatan: Semua data asas permainan, seperti tahap permainan anda, jumlah wang maya dan sumber, kemajuan dalam permainan, dan tetapan, disimpan hanya secara tempatan pada peranti anda sendiri, bukan pada pelayan kami. Kami не mempunyai akses jauh ke data ini.
  • Data Log Masuk Google (Untuk Ciri Sandaran): Apabila anda menggunakan ciri untuk membuat sandaran dan memulihkan kemajuan permainan anda ke Google Drive, anda akan diminta untuk mengakses maklumat profil asas akaun Google anda (khususnya alamat e-mel anda) untuk mengesahkan identiti anda. Maklumat ini hanya digunakan untuk mengesahkan bahawa sandaran itu milik anda dan tidak diproses atau disimpan untuk sebarang tujuan lain.
  • Data Iklan: Aplikasi kami mungkin memaparkan iklan melalui rangkaian iklan pihak ketiga, seperti Google AdMob. Rangkaian ini mungkin mengumpul data tanpa nama, seperti ID pengiklanan peranti anda, untuk menyediakan iklan yang disesuaikan dengan minat anda. Pengumpulan dan pemprosesan data ini tertakluk pada dasar privasi rangkaian iklan masing-masing.

2. Penggunaan Data

Kami menggunakan data yang kami kumpul untuk tujuan berikut:

  • Menyediakan fungsi permainan dan menyimpan kemajuan permainan anda.
  • Secara pilihan membolehkan anda membuat sandaran dan memulihkan data permainan anda ke akaun Google Drive anda.
  • Menyokong kos pembangunan dan penyelenggaraan aplikasi melalui pengiklanan dalam aplikasi.

3. Keselamatan Data

Kami sangat mementingkan keselamatan data pengguna. Data tempatan yang disimpan pada peranti anda dilindungi oleh langkah keselamatan standard sistem operasi anda. Ciri sandaran Google Drive menggunakan infrastruktur keselamatan Google sendiri.

4. Dasar dan Garis Panduan Pemadaman Data

Mengikut dasar Google Play, kami menawarkan semua pengguna kami hak untuk memadamkan data mereka. Kerana aplikasi kami tidak menggunakan sistem akaun berasaskan pelayan, "memadamkan akaun" bermakna menetapkan semula secara kekal semua data permainan tempatan dan kemajuan pada peranti anda.

Untuk Memadamkan Data Anda, Sila Ikuti Langkah-langkah Berikut:

  1. Buka aplikasi "Mega Factory Tycoon: Grid Pengeluaran".
  2. Pergi ke Tetapan (biasanya diwakili oleh ikon gear) dari menu utama atau skrin permainan.
  3. Dalam menu Tetapan, cari pilihan bertajuk "Tetapkan Semula Permainan" atau "Padam Semua Data".
  4. Apabila anda mengetik pilihan ini, tetingkap pengesahan akan muncul menyatakan bahawa tindakan itu adalah kekal dan tidak boleh dibatalkan.
  5. Sebaik sahaja anda mengesahkan tindakan itu, semua kemajuan permainan, sumber, tetapan, dan data lain pada peranti anda akan dipadamkan secara kekal. Aplikasi akan kembali ke keadaan asalnya, seolah-olah ia baru sahaja dipasang.

Nota Penting: Tindakan ini tidak akan memadamkan sandaran anda di Google Drive secara automatik. Jika anda ingin mengurus sandaran anda secara berasingan, anda boleh mengalih keluar akses aplikasi dari tetapan Google Drive anda.
