Kebijakan Privasi dan Manajemen Data untuk "Mega Factory Tycoon: Jaringan Produksi"

Pembaruan Terakhir: [17.10.2025]

Kebijakan privasi ini menjelaskan bagaimana aplikasi seluler "Mega Factory Tycoon: Jaringan Produksi" mengumpulkan, menggunakan, dan mengelola data pengguna.

1. Data yang Kami Kumpulkan

Aplikasi kami hanya mengumpulkan data terbatas yang diperlukan untuk menyediakan dan meningkatkan pengalaman bermain game.

  • Data Game Lokal: Semua data game dasar, seperti level game Anda, jumlah uang virtual dan sumber daya, progres dalam game, dan pengaturan, disimpan hanya secara lokal di perangkat Anda sendiri, bukan di server kami. Kami tidak memiliki akses jarak jauh ke data ini.
  • Data Login Google (Untuk Fitur Cadangan): Saat Anda menggunakan fitur untuk mencadangkan dan memulihkan progres game Anda ke Google Drive, Anda akan diminta untuk mengakses informasi profil dasar akun Google Anda (khususnya alamat email Anda) untuk memverifikasi identitas Anda. Informasi ini hanya digunakan untuk mengonfirmasi bahwa cadangan tersebut milik Anda dan tidak diproses atau disimpan untuk tujuan lain.
  • Data Iklan: Aplikasi kami dapat menampilkan iklan melalui jaringan iklan pihak ketiga, seperti Google AdMob. Jaringan ini dapat mengumpulkan data anonim, seperti ID iklan perangkat Anda, untuk menyediakan iklan yang disesuaikan dengan minat Anda. Pengumpulan dan pemrosesan data ini tunduk pada kebijakan privasi masing-masing jaringan iklan.

2. Penggunaan Data

Kami menggunakan data yang kami kumpulkan untuk tujuan berikut:

  • Menyediakan fungsionalitas game dan menyimpan progres game Anda.
  • Secara opsional memungkinkan Anda untuk mencadangkan dan memulihkan data game Anda ke akun Google Drive Anda.
  • Mendukung biaya pengembangan dan pemeliharaan aplikasi melalui iklan dalam aplikasi.

3. Keamanan Data

Kami sangat mementingkan keamanan data pengguna. Data lokal yang disimpan di perangkat Anda dilindungi oleh langkah-langkah keamanan standar sistem operasi Anda. Fitur cadangan Google Drive menggunakan infrastruktur keamanan Google sendiri.

4. Kebijakan dan Pedoman Penghapusan Data

Sesuai dengan kebijakan Google Play, kami menawarkan semua pengguna kami hak untuk menghapus data mereka. Karena aplikasi kami не menggunakan sistem akun berbasis server, "menghapus akun" berarti mengatur ulang secara permanen semua data game lokal dan progres di perangkat Anda.

Untuk Menghapus Data Anda, Silakan Ikuti Langkah-Langkah Berikut:

  1. Buka aplikasi "Mega Factory Tycoon: Jaringan Produksi".
  2. Masuk ke Pengaturan (biasanya diwakili oleh ikon roda gigi) dari menu utama atau layar game.
  3. Di menu Pengaturan, cari opsi berjudul "Atur Ulang Game" atau "Hapus Semua Data".
  4. Saat Anda mengetuk opsi ini, jendela konfirmasi akan muncul yang menyatakan bahwa tindakan tersebut bersifat permanen dan tidak dapat dibatalkan.
  5. Setelah Anda mengonfirmasi tindakan tersebut, semua progres game, sumber daya, pengaturan, dan data lain di perangkat Anda akan dihapus secara permanen. Aplikasi akan kembali ke keadaan semula, seolah-olah baru pertama kali diinstal.

Catatan Penting: Tindakan ini tidak akan secara otomatis menghapus cadangan Anda di Google Drive. Jika Anda ingin mengelola cadangan Anda secara terpisah, Anda dapat menghapus akses aplikasi dari pengaturan Google Drive Anda.

5. Privasi Anak-Anak

Aplikasi kami tidak ditujukan untuk anak-anak di bawah usia 13 tahun, dan kami tidak dengan sengaja mengumpulkan informasi pribadi dari mereka.
